Reminder

Download

This is a very, very, very simple program that does one thing very well: reminds you of important occasions in the coming two weeks. Birthdays, bills, deadlines and anniversaries alike all become memorable once this small software is inside your 'startup' folder. The program is given as-is and is absolutely free, no liabilities implied. This is version 1.0 written by David Talby; an example data file for the program is here.
