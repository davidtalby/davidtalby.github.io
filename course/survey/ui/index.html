User Interface Track

Ill-designed interfaces are reportedly the biggest problem for computer users today, now that hardware and communication are no longer a big constraint. The UI is often the most critical factor to determine success of a software product, since it often implements itself using the same technologies and algorithms as its competitors. Enormous efforts have been put into developing better interfaces, and making it easier for developers to use them. This track covers the major advances for different kinds of applications.

 

Main Lectures of this Track

1. DirectX

given by Shimrit David, Ayelet Elbaz and Eduardo Vainroj

The library on which graphics, video and games are built in the Windows platform.

The two demos are at the lecturers' discretion, but must be non-trivial and demonstrate different features.

2. DHTML, CSS and Behaviors

given by Sagiv Friemark, Shiran Levi and Ziv Balshai

These three new standards greatly improve the flexibility and ease of writing attractive web sites. The lecture presents the these standards, gives examples of the code used to implement cool features with them, explains how they work together and complement each other, and compares the best tools for developers that write sites with them.

The two demos must show practical usage of all three technologies, in any combination.

3. Flash

given by Orit Mouki, Asaf Eisner and Yaron Zalikovits

Flash is a technology for creating high impact web sites, alive with animations, sounds, innovative interfaces and amazing graphics. Flash integrates with vector art, mp3 audio, sound and images to produce impressive web sites. This lecture demonstrates what can be done, and what development kits and environments are available.

The two demos are at the lecturers' description, and as usual should demonstrate two different features of the technology, in a non-trivial setting.

4. Advances in Windows UI

given by Eran Amzelag and Shlomi Armon

This lecture covers the fancy UI enhancements available in the Windows platform today: Sophisticated grid with sorting and grouping, tree views with drag-n-drop, docking, Office 2000 style menus and toolbars, shaped forms, transparent forms, data-aware components and scripting. The lecture should also discover how easily these are available to developers today, using Delphi freeware and shareware components. 

The two demos must include one MDI editor application that employs Office-2000 style menus and toolbars as well as docking and transparency, and another one that demonstrates data-aware components over an in-memory database.

 Short Lectures of this Track

The short lectures, as usual, are about the leading tools, but ones used in smaller market niches.

1. Swing

given by: Yaron Basson, Yosi Cohen, Nariman Cohen

Overview the Java UI library, which is platform independent and has an elegant object-oriented design.

2. WinForms and WebForms

given by: Available

Microsoft's dot-Net vision for Windows-based and Web-based user interfaces. The lecture compares what will be available for each of them, how form inheritance works in VB7, and how the user interface is accessed in dot-Net from both form types.

3. VRML

given by: Amir Levi, Michel Moshayoff and Kobi Levi

The Virtual Reality Modeling Language works over the Internet and support cool 3D modeling and animation. Cover the main features, popular development environments, and give a demo and its source code.

4. UIs for People with Disabilities

given by: Hani, Anna and Hadas

Industrial offerings of hardware and software for the blind, deaf, handicapped, and very old or very young users.

 

Projects of this Track

Projects in this track should create particularly smart and beautiful user interfaces that make large improvements in the way people work. A three-dimensional Internet browser, an all-in-one-grid database or a new improved set of Delphi or COM components are worthy projects.
